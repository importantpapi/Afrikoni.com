# AFRIKONI BRAND GUIDELINES
## Trade. Trust. Thrive.

## 1. BRAND OVERVIEW

### Mission Statement

**Afrikoni is the infrastructure layer for African B2B trade** - connecting verified suppliers and buyers across 54 African countries with trust, security, and efficiency.

### Brand Promise

We enable African businesses to **trade** across borders with **trust**, so they can **thrive** together.

### Core Values

| Value | Meaning |
|-------|---------|
| **Trust** | Verified suppliers, secure payments, transparent transactions |
| **Unity** | Connecting 54 countries as one marketplace |
| **Empowerment** | Enabling African businesses to compete globally |
| **Integrity** | Fair dealings, honest practices, ethical trade |
| **Innovation** | Technology solving real African trade problems |

## 4. TYPOGRAPHY

### Primary Typeface

**Headings: Playfair Display** (or similar serif)
- Elegant, trustworthy, premium feel
- Use for: H1, H2, H3, logo wordmark

**Body: Inter** (or similar sans-serif)
- Clean, readable, modern
- Use for: Body text, UI elements, buttons

### Type Scale

```
H1:     48px / 56px line-height / Bold
H2:     36px / 44px line-height / Bold
H3:     28px / 36px line-height / SemiBold
H4:     24px / 32px line-height / SemiBold
H5:     20px / 28px line-height / Medium
H6:     16px / 24px line-height / Medium

Body:   16px / 24px line-height / Regular
Small:  14px / 20px line-height / Regular
Tiny:   12px / 16px line-height / Regular

Button: 16px / 24px line-height / SemiBold
Label:  14px / 20px line-height / Medium
```

### Typography Examples

```
TRADE. TRUST. THRIVE.
Playfair Display Bold, 14px, Letter-spacing: 3px

Connecting African Businesses
Playfair Display SemiBold, 36px

Afrikoni is the B2B marketplace for verified
African suppliers and buyers.
Inter Regular, 16px
```

## 5. IMAGERY GUIDELINES

### Photography Style

| Do | Don't |
|----|-------|
| Real African business people | Stock photos of generic offices |
| Authentic marketplaces and trade | Overly staged or Western settings |
| Warm, golden-hour lighting | Cold, clinical lighting |
| Products being made/traded | Random product shots |
| Human connection and handshakes | Isolated individuals |

### Image Treatment

- Apply warm color grading to match brand palette
- Subtle golden overlay (10-20% opacity) for cohesion
- High contrast for visibility
- Focus on people and products, not technology

### Iconography

- **Style:** Outlined, 2px stroke
- **Corners:** Slightly rounded (2px radius)
- **Color:** Afrikoni Gold on dark, Afrikoni Brown on light
- **Size:** 24px standard, 32px for emphasis

## 6. VOICE & TONE

### Brand Voice

| Attribute | Description |
|-----------|-------------|
| **Confident** | We know African trade. We've built the solution. |
| **Warm** | We're partners, not just a platform. |
| **Direct** | Clear communication, no jargon. |
| **Empowering** | We enable success, not dependency. |
| **Pan-African** | We celebrate all 54 nations. |

### Tone Examples

**DO write:**
> "Find verified suppliers across Africa in minutes, not months."

> "Your payment is protected until you confirm delivery."

> "Join 500+ businesses trading with confidence on Afrikoni."

**DON'T write:**
> "Our revolutionary AI-powered blockchain platform..."

> "Afrikoni leverages synergies to optimize B2B workflows..."

> "Click here to learn more about our amazing features!"

### Key Messages

1. **Trust Message:**
   > "Every supplier verified. Every payment protected."

2. **Scale Message:**
   > "One platform. 54 countries. Unlimited opportunity."

3. **Empowerment Message:**
   > "Built by Africa, for Africa, to connect the world."

4. **Simplicity Message:**
   > "Trade across borders as easily as trading next door."
