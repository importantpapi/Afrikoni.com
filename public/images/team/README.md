# Team Images

Place team member photos here.

## File Naming Convention
- Use lowercase with hyphens: `youba-simao-thiam.jpg`
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

## Image Requirements
- **Recommended size:** 400x400px minimum (square format works best)
- **Format:** JPG or PNG
- **Optimized for web:** Compressed for fast loading
- **Quality:** High resolution for crisp display (will be displayed at 144-160px on desktop)

## Current Team Members

## Image Display
- Images are displayed in circular frames with gold borders
- Founder image has enhanced styling with larger size and premium shadow effects
- Images are optimized with lazy loading and responsive sizing
